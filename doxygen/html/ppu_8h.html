◆ StateSpriteEvaluation

enum StateSpriteEvaluation

State for Sprite Evaluation algorithm.

Enumerator
STATE_COPY_Y 

Y-coordonate copy from OAM to SOAM

STATE_COPY_REMAINING 

Copy remaining 3 byte from OAM to SOAM

STATE_OVERFLOW 

8 sprites has been written in SOAM

STATE_OVERFLOW_REMAINING 

Sprite overflow has occured

STATE_WAIT 

All sprites has been evaluated

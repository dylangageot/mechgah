Detailed Description

Register use to communicate with PPU, APU and joystick.

The structure is composed of two arrays that represent bank in memory map :

  • bank1 : @0x2000, registers for PPU
  • bank2 : @0x4000, registers for APU and joystick Flags array are used to know if data was read or written
